# Descripcion # 
Este es un gimnasio urbano el cual tiene una paleta de colores negro, rojo y gris. La página cuenta con una hero section donde esta una breve descripción además de contar con planes variados cuenta con información propia una sección para pedir información llenando un form y un footer.


Objetivo de la landing page
La página representa una pagina informativa y para inscribirse a un plan del gimnasio esta dirigida a los usuarios que quieren asistir al gimnasio y se espera que el visitante vea la información presentada y se anime a inscribirse a uno de los planes.
